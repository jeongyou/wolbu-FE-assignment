## 과제 체크리스트

## 1. 회원 가입 체크리스트

### 화면 및 흐름

- [ ] `/signup` 경로에서 회원가입 화면이 렌더링된다
- [ ] 회원가입 완료 시 `/lectures`로 이동한다

### 입력 필드

- [ ] 이름 입력 필드가 있다
- [ ] 이메일 입력 필드가 있다
- [ ] 휴대폰 번호 입력 필드가 있다
- [ ] 비밀번호 입력 필드가 있다
- [ ] 회원 유형 선택 필드가 있다 (수강생 / 강사)

### 비밀번호 검증

- [ ] 비밀번호 길이가 6자 이상 10자 이하인지 검증한다
- [ ] 영문 소문자, 대문자, 숫자 중 최소 2가지 이상 포함하는지 검증한다
- [ ] 비밀번호 조건 안내 문구가 화면에 표시된다
- [ ] 조건 미충족 시 에러 메시지가 표시된다

### 제출

- [ ] 모든 필수 값 입력 후 가입하기 버튼을 클릭할 수 있다
- [ ] 유효하지 않은 입력이 있을 경우 가입이 되지 않는다
- [ ] 가입 성공 시 강의 목록 화면으로 이동한다

## 2. 강의 개설 체크리스트

### 접근 권한

- [ ] 강사 회원만 강의 등록 화면에 접근할 수 있다
- [ ] 수강생 회원은 강의 등록이 제한된다

### 화면 및 흐름

- [ ] 강의 목록 화면에서 강의 개설 버튼이 노출된다 (강사만)
- [ ] 강의 개설 버튼 클릭 시 `/lectures/new`로 이동한다
- [ ] 강의 등록 완료 시 `/lectures`로 이동한다

### 입력 필드

- [ ] 강의명 입력 필드가 있다
- [ ] 최대 수강 인원 입력 필드가 있다
- [ ] 가격 입력 필드가 있다

### 제출

- [ ] 모든 필수 값 입력 후 등록하기 버튼을 클릭할 수 있다
- [ ] 입력 값이 없거나 잘못된 경우 등록이 되지 않는다

## 3. 수강 신청 체크리스트

### 강의 목록

- [ ] `/lectures` 경로에서 강의 목록이 렌더링된다
- [ ] 강의 목록이 카드 리스트 형태로 표시된다
- [ ] 강의 목록에 무한 스크롤이 적용된다
- [ ] 스크롤 시 자연스럽게 다음 강의가 로드된다

### 강의 아이템 표시 정보

- [ ] 강의명이 표시된다
- [ ] 가격이 표시된다
- [ ] 강사명이 표시된다
- [ ] 현재 신청자 수가 표시된다
- [ ] 최대 수강 인원 수가 표시된다

### 정렬

- [ ] 최근 등록순 정렬이 가능하다
- [ ] 신청자 많은 순 정렬이 가능하다
- [ ] 신청률 높은 순 정렬이 가능하다
- [ ] 신청률은 `(신청자 수 / 최대 수강 인원)`으로 계산된다

### 선택 및 신청

- [ ] 강의 아이템을 선택할 수 있다 (checkbox)
- [ ] 여러 강의를 동시에 선택할 수 있다
- [ ] 선택된 강의가 없으면 수강 신청 버튼이 비활성화된다
- [ ] 선택된 강의가 있으면 수강 신청 버튼이 활성화된다
- [ ] 강사 회원도 수강 신청이 가능하다
